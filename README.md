# A&A Law Firm Landing Page

Landing page desarrollada en **Next.js + TypeScript + Tailwind CSS + Framer Motion** para despliegue en **GitHub + Vercel**.

## Incluye

- Hero premium responsive con imagen diferente para móvil y desktop.
- Preloader enfocado en el logo.
- Sección completa de servicios legales.
- CTA a WhatsApp.
- Formulario que prepara el mensaje y lo envía a WhatsApp.
- FAQ orientado a SEO inicial.
- Metadata SEO, sitemap y robots.
- Assets visuales y logo incluidos dentro del proyecto.

## Instalación local

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm start
```

## Despliegue en Vercel

1. Suba este proyecto a un repositorio nuevo en GitHub.
2. Entre a Vercel.
3. Importe el repositorio.
4. Vercel detectará Next.js automáticamente.
5. Haga deploy.

## Notas

- El correo usado en la interfaz es `info@aalawfirmpa.com` como referencia visual.
- El formulario actual envía la información hacia WhatsApp. Si luego desea envío real por email, se puede conectar a Resend, Formspree, EmailJS o API propia.
